# ccms-svg-icons

### 依赖
依赖 AngularJS 1.5.8

### 安装

```
npm install ccms-svg-icons --save;
```

### 使用

```js
// 单独引入你需要使用的 icon 组件
import iconWechat from 'ccms-svg-icons/svgs/iconWechat'
```

```html
<icon-wechat></icon-wechat>
```

### 发布

注意: 将新的 iconfont.js 拷贝到 svg-source 目录
```
npm run release
```
